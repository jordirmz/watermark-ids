Watermark IDs
==========

Watermark IDs es una herramienta para proteger imágenes antes de compartirlas. Funciona completamente en el navegador y no envía ningún dato a servidores externos.

Funciones principales
---------------------

*   Censura con pincel pintando directamente sobre la imagen.
    
*   Recorta cualquier parte de la imagen.
    
*   Añade una marca de agua personalizada.

*   Funciones para ocultar automáticamente datos del DNI español (Beta).
    
*   Procesamiento 100% local, sin conexión a internet.
    
*   Descarga la imagen final en formato PNG.
    

Privacidad
----------

*   Ninguna imagen se sube, envía o almacena en servidores.
    
*   Todo el procesamiento ocurre en el dispositivo del usuario.
    

Cómo usarlo
-----------

1.  Abre `index.html` en tu navegador.
    
2.  Sube una imagen.
    
3.  Censura o recorta las partes necesarias.
    
4.  Añade una marca de agua.
    
5.  Descarga la imagen procesada.
